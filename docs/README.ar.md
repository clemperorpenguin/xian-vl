# Xian-MAGE — مساعد الرؤية واللغة في الوقت الفعلي لبيئات الألعاب 🧙‍♂️🎮🤖

Xian-MAGE هو واجهة عرض داخل اللعبة (HUD) ومساعد دائم على سطح المكتب يعمل في الوقت الفعلي لأنظمة لينكس (Wayland). مدعوم بالخلفية **[خادم Lemonade](https://lemonade-server.ai/)**، يقوم بعرض التعرف الضوئي على الحروف (OCR) في الوقت الفعلي، والترجمة، وأدوات تصور النقر بالتوجيه البصري، والدردشة التفاعلية مباشرة فوق بيئات الألعاب النشطة.

نظرًا لأن عملية استدلال النموذج تُنسّق عبر Lemonade، **يدعم MAGE التنفيذ المسرّع بواسطة Vulkan، ويعمل بسلاسة على وحدات معالجة الرسومات AMD Radeon™ وغيرها من المسرعات.**

## 🌟 الميزات الأساسية (قابلة للتشغيل بالكامل)

- **تسريع Vulkan / AMD GPU**: تنفيذ نموذج الرؤية واللغة بزمن انتقال منخفض مدعوم بتسريع الأجهزة الخلفية لوحدة معالجة الرسومات.
- **واجهة تراكب سطح المكتب مع تمرير النقرات**: نوافذ تراكب شفافة مبنية على PyQt6 تعرض النص المترجم مباشرة فوق واجهات اللعبة والحوارات، مع البقاء غير مرئية تمامًا لمدخلات الماوس.
- **مفاتيح التشغيل السريع العامة في Wayland وقائمة OSD للأوامر**: تشغيل الترجمة وتكوينات OSD والأشرطة الجانبية بسلاسة باستخدام مفاتيح تشغيل سريع قابلة للتخصيص على مستوى النظام.
- **وضع الحوار (التشغيل التلقائي للروايات المرئية / ألعاب RPG القصصية)**: قفل منطقة على الشاشة، والترجمة تلقائيًا، والمتابعة/تحديث للترجمات مضمنة بنقرة ماوس بسيطة.
- **تمييز أهداف التوجيه البصري**: اسأل المساعد *"أين أنقر؟"* أو *"أين المخرج؟"* وشاهده يبرز الإحداثيات الفعلية الدقيقة على شاشتك.
- **الوضع السينمائي (الترجمة الصوتية السياقية)**: يدمج بسلاسة تحليل رؤية التقاط الشاشة مع ترجمة تشغيل الصوت.
- **قاموس CC-CEDICT المحلي**: مرر المؤشر فوق أي فقاعة ترجمة واضغط على `Alt` للحصول على تحليل محلي آمن على مستوى خيوط المعالجة للأحرف الصينية والبنيين (Pinyin) والتعريفات.

## 🛠️ البدء (عميل MAGE)

### المتطلبات

- **لينكس مع Wayland** (يدعم X11 أيضًا؛ تتطلب اختصارات المفاتيح العامة مدخلات `evdev`)
- **أذونات المستخدم**: يجب أن يكون المستخدم الخاص بك في مجموعة `input` لالتقاط مفاتيح التشغيل السريع العامة (`sudo usermod -aG input $USER` ثم قم بتسجيل الخروج/الدخول)
- **خادم Lemonade**: نسخة عاملة من خادم Lemonade (يمكن الوصول إليها افتراضيًا عبر `http://localhost:13305`)

### الإعداد السريع (لينكس)

استنسخ المستودع وقم بتشغيل سكريبت التهيئة — سيقوم بتثبيت [`uv`](https://docs.astral.sh/uv/)، ومزامنة جميع التبعيات، وتشغيل MAGE تلقائيًا:

```bash
git clone https://github.com/clemperorpenguin/xian-vl.git
cd xian-vl
./mage.sh
```

لإضافة MAGE إلى قائمة تطبيقات سطح المكتب:

```bash
./mage.sh --install
```

لإضافة MAGE إلى قائمتك **و** تثبيت تبعيات النظام تلقائيًا، وبناء خادم Lemonade المضمن من المصدر، وسحب نموذج الرؤية واللغة الافتراضي:

```bash
./mage.sh --install --build
```

لإزالة إدخال سطح المكتب والأيقونة:

```bash
./mage.sh --uninstall
```

### الإصدارات الجاهزة (ويندوز و ماك)

إذا كنت تستخدم ويندوز أو ماك، أو تفضل عدم البناء من المصدر، فقم بتنزيل الحزم الجاهزة من صفحة [إصدارات GitHub](https://github.com/clemperorpenguin/xian-vl/releases).

تأتي الإصدارات بنسختين:
- **Lite (الخفيف)**: نسخة مستقلة خفيفة الوزن. تتطلب الاتصال بخادم Lemonade خارجي قيد التشغيل.
- **Full (الكامل)**: مدمج مع خادم `lemond` المضمن، والذي يبدأ ويتوقف تلقائيًا عند تشغيل التطبيق.

#### ويندوز
1. قم بتنزيل `mage-client-Windows-x86-64-lite.zip` أو `mage-client-Windows-x86-64-full.zip`.
2. فك ضغط الأرشيف.
3. انقر نقرًا مزدوجًا على `mage-client.exe` للتشغيل.

#### ماك
1. قم بتنزيل `mage-client-MacOS-ARM64-lite.dmg` أو `mage-client-MacOS-ARM64-full.dmg` (أو ما يعادلها بصيغة ZIP).
2. انقر نقرًا مزدوجًا على ملف DMG واسحب `mage-client.app` إلى مجلد **التطبيقات**.
3. افتح التطبيق وقم بتشغيله.
   > [!NOTE]
   > نظرًا لأن التطبيق لم يتم توثيقه/توقيعه بواسطة آبل، ستقوم بوابة الحماية (Gatekeeper) بحظره مع تحذير يفيد بأن التطبيق "تالف ولا يمكن فتحه". يمكنك إصلاح ذلك بسهولة عن طريق تشغيل الأمر التالي في الطرفية:
   > ```bash
   > xattr -cr /Applications/mage-client.app
   > ```

### الإعداد اليدوي (جميع المنصات)

إذا كنت تفضل إدارة البيئة بنفسك:

```bash
git clone https://github.com/clemperorpenguin/xian-vl.git
cd xian-vl
uv sync --all-packages
uv run --package mage-client mage
```

### عناصر التحكم
- **فتح قائمة الإجراءات / OSD** — نقر مزدوج على `Shift` (المفتاح الرئيسي الافتراضي)
- **تفعيل التقاط الشاشة** — `C` في قائمة الإجراءات (حدد منطقة الشاشة، ثم اختر الترجمة / الحوار / الدردشة)
- **تبديل الشريط الجانبي للدردشة** — `A` في قائمة الإجراءات
- **الترجمة للدردشة (الإدخال)** — `T` في قائمة الإجراءات
- **لوحة الإعدادات** — `S` في قائمة الإجراءات

## 📁 البنية والمشاريع الفرعية

يحتوي المستودع الموحد على عميل MAGE الأساسي الجاهز للإنتاج بالإضافة إلى هياكل تجريبية للمشاريع المصاحبة:

```
├── apps/
│   ├── mage-client/      # 🧙‍♂️ تطبيق واجهة العرض داخل اللعبة الرئيسي المعتمد، المبني على PyQt6
│   ├── nate/             # 📱 قارئ OCR وتطبيق قاموس مرافق لأندرويد (تجريبي)
│   ├── masha-extension/  # 🌐 امتداد متصفح لترجمة النص المحدد (تجريبي)
│   ├── lore-client/      # 📜 أداة سطر أوامر لبناء موسوعة معرفية باستخدام RAG (تجريبي)
│   └── luduan-client/    # 🦤 أداة سطر أوامر لترجمة EPUB والكتب الصوتية (تجريبي)
└── packages/
    ├── xian-vl/          # ⚙️ محرك تنسيق LLM/ASR الأساسي ومديرو السياق
    └── shared-types/     # 📦 النماذج القياسية والثوابت والأنواع المشتركة
```

> [!WARNING]
> **قيود ASR / الصوت**: إرسال تحميلات البث الصوتي المباشر إلى Lemonade معطّل حاليًا بسبب قيود الخلفية من جانب الخادم. نتيجة لذلك، تم تعطيل ميزات ترجمة الكلام المباشر (مثل **وضع الغارة**). جميع ميزات التعرف الضوئي على الحروف البصري، وترجمة النص، والقاموس، والتوجيه البصري في الدردشة تعمل بكامل طاقتها.
