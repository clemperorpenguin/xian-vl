# Xian-MAGE — गेमिंग वातावरण के लिए रियल-टाइम विज़न-लैंग्वेज असिस्टेंट 🧙‍♂️🎮🤖

Xian-MAGE लिनक्स (Wayland) के लिए एक रियल-टाइम, पर्सिस्टेंट डेस्कटॉप गेमिंग HUD और असिस्टेंट है। **[Lemonade Server](https://lemonade-server.ai/)** बैकएंड द्वारा संचालित, यह रियल-टाइम OCR, अनुवाद, विज़ुअल ग्राउंडिंग क्लिक विज़ुअलाइज़र, और इंटरैक्टिव संवादात्मक चैट को सीधे सक्रिय गेमिंग वातावरण के शीर्ष पर ओवरले करता है।

चूंकि इनफरेंस Lemonade के माध्यम से ऑर्केस्ट्रेट किया जाता है, **MAGE Vulkan-त्वरित निष्पादन का समर्थन करता है, और AMD Radeon™ GPUs और अन्य एक्सेलेरेटरों पर सुचारू रूप से चलता है।**

## 🌟 मुख्य विशेषताएं (पूर्ण रूप से चालू)

- **Vulkan / AMD GPU एक्सेलेरेटेड**: बैकएंड GPU हार्डवेयर एक्सीलरेशन द्वारा संचालित, कम विलंबता वाली विज़न-लैंग्वेज मॉडल निष्पादन।
- **क्लिक-थ्रू डेस्कटॉप ओवरले**: पारदर्शी PyQt6 ओवरले विंडोज़ जो गेम HUD और डायलॉग के ठीक ऊपर अनुवादित टेक्स्ट प्रदर्शित करती हैं, जबकि माउस इनपुट के लिए पूरी तरह से अदृश्य रहती हैं।
- **Wayland ग्लोबल हॉटकी और कमांड OSD**: कस्टमाइज़ करने योग्य सिस्टम-वाइड लीडर हॉटकी का उपयोग करके अनुवाद, OSD कॉन्फ़िगरेशन और साइडबार को निर्बाध रूप से ट्रिगर करें।
- **डायलॉग मोड (ऑटोप्ले VNs / स्टोरी RPGs)**: स्क्रीन के एक क्षेत्र पर लॉक करें, स्वचालित रूप से अनुवाद करें, और एक साधारण माउस क्लिक के साथ इनलाइन अनुवाद को आगे बढ़ाएं/रिफ्रेश करें।
- **विज़ुअल ग्राउंडिंग टारगेट हाइलाइटिंग**: असिस्टेंट से पूछें *"मुझे कहां क्लिक करना चाहिए?"* या *"बाहर जाने का रास्ता कहां है?"* और देखें कि यह आपकी स्क्रीन पर सटीक भौतिक निर्देशांकों को कैसे हाइलाइट करता है।
- **सिनेमैटिक मोड (कंटेक्स्टुअल वॉइस ट्रांसलेशन)**: स्क्रीन कैप्चर विज़न एनालिसिस को ऑडियो प्लेबैक अनुवाद के साथ निर्बाध रूप से जोड़ता है।
- **लोकल CC-CEDICT डिक्शनरी**: किसी भी अनुवाद बबल पर तुरंत होवर करें और चीनी अक्षरों, पिनयिन और परिभाषाओं के लोकल थ्रेड-सेफ पार्सिंग ब्रेकडाउन के लिए `Alt` दबाएं।

## 🛠️ शुरुआत करें (MAGE क्लाइंट)

### आवश्यकताएं

- **Wayland के साथ लिनक्स** (X11 भी समर्थित है; ग्लोबल की-बाइंडिंग के लिए `evdev` इनपुट की आवश्यकता होती है)
- **उपयोगकर्ता अनुमतियां**: ग्लोबल हॉटकी कैप्चरिंग के लिए आपका उपयोगकर्ता `input` ग्रुप में होना चाहिए (`sudo usermod -aG input $USER` और लॉग आउट/इन करें)
- **Lemonade Server**: एक चल रहा Lemonade Server इंस्टेंस (डिफ़ॉल्ट रूप से `http://localhost:13305` पर एक्सेस किया जा सकता है)

### त्वरित सेटअप (लिनक्स)

रिपॉजिटरी को क्लोन करें और बूटस्ट्रैप स्क्रिप्ट चलाएं — यह [`uv`](https://docs.astral.sh/uv/) इंस्टॉल करता है, सभी डिपेंडेंसीज़ को सिंक करता है, और MAGE को स्वचालित रूप से लॉन्च करता है:

```bash
git clone https://github.com/clemperorpenguin/xian-vl.git
cd xian-vl
./mage.sh
```

MAGE को अपने डेस्कटॉप एप्लिकेशन मेनू में जोड़ने के लिए:

```bash
./mage.sh --install
```

MAGE को अपने मेनू में जोड़ने के लिए **और** स्वचालित रूप से सिस्टम डिपेंडेंसीज़ इंस्टॉल करने, सोर्स से एम्बेडेबल Lemonade सर्वर बनाने, और डिफ़ॉल्ट विज़न-लैंग्वेज मॉडल खींचने के लिए:

```bash
./mage.sh --install --build
```

डेस्कटॉप एंट्री और आइकन को हटाने के लिए:

```bash
./mage.sh --uninstall
```

### प्री-बिल्ट रिलीज़ (Windows और macOS)

यदि आप Windows या macOS चला रहे हैं, या सोर्स से बिल्ड नहीं करना चाहते, तो [GitHub Releases](https://github.com/clemperorpenguin/xian-vl/releases) पेज से प्री-बिल्ट पैकेज डाउनलोड करें।

रिलीज़ दो वेरिएंट में आती हैं:
- **Lite**: स्टैंडअलोन हल्का वर्जन। बाहर से चल रहे Lemonade Server से कनेक्ट करने की आवश्यकता होती है।
- **Full**: एम्बेडेड `lemond` सर्वर के साथ बंडल किया गया, जो एप्लिकेशन चलने पर स्वचालित रूप से शुरू और बंद होता है।

#### Windows
1. `mage-client-Windows-x86-64-lite.zip` या `mage-client-Windows-x86-64-full.zip` डाउनलोड करें।
2. आर्काइव को एक्सट्रैक्ट करें।
3. चलाने के लिए `mage-client.exe` पर डबल-क्लिक करें।

#### macOS
1. `mage-client-MacOS-ARM64-lite.dmg` या `mage-client-MacOS-ARM64-full.dmg` (या ZIP समकक्ष) डाउनलोड करें।
2. DMG पर डबल-क्लिक करें और `mage-client.app` को अपने **एप्लिकेशन** डायरेक्टरी में ड्रैग करें।
3. एप्लिकेशन को खोलें और चलाएं।
   > [!NOTE]
   > चूंकि एप्लिकेशन को Apple द्वारा नोटरीकृत/साइन नहीं किया गया है, Gatekeeper इसे "क्षतिग्रस्त है और खोला नहीं जा सकता" चेतावनी के साथ ब्लॉक कर देगा। आप अपने टर्मिनल में निम्नलिखित कमांड चलाकर इसे आसानी से ठीक कर सकते हैं:
   > ```bash
   > xattr -cr /Applications/mage-client.app
   > ```

### मैनुअल सेटअप (सभी प्लेटफॉर्म)

यदि आप स्वयं एनवायरनमेंट प्रबंधित करना पसंद करते हैं:

```bash
git clone https://github.com/clemperorpenguin/xian-vl.git
cd xian-vl
uv sync --all-packages
uv run --package mage-client mage
```

### नियंत्रण
- **एक्शन मेनू / OSD खोलें** — `Shift` को डबल-टैप करें (डिफ़ॉल्ट लीडर की)
- **स्क्रीन कैप्चर ट्रिगर करें** — एक्शन मेनू `C` (स्क्रीन क्षेत्र चुनें, फिर अनुवाद / डायलॉग / चैट)
- **चैट साइडबार टॉगल करें** — एक्शन मेनू `A`
- **चैट के लिए अनुवाद करें (इनपुट)** — एक्शन मेनू `T`
- **सेटिंग्स पैनल** — एक्शन मेनू `S`

## 📁 आर्किटेक्चर और स्पोक प्रोजेक्ट्स

मोनोरेपो में कोर प्रोडक्शन-रेडी MAGE क्लाइंट के साथ-साथ प्रयोगात्मक साथी स्कैफोल्ड शामिल हैं:

```
├── apps/
│   ├── mage-client/      # 🧙‍♂️ मुख्य सत्यापित, PyQt6-आधारित गेमिंग HUD एप्लिकेशन
│   ├── nate/             # 📱 एंड्रॉइड कंपैनियन OCR रीडर और डिक्शनरी (प्रयोगात्मक)
│   ├── masha-extension/  # 🌐 ब्राउज़र एक्सटेंशन सिलेक्शन ट्रांसलेटर (प्रयोगात्मक)
│   ├── lore-client/      # 📜 RAG नॉलेज विकी बिल्डर CLI (प्रयोगात्मक)
│   └── luduan-client/    # 🦤 EPUB अनुवाद और ऑडियोबुक CLI (प्रयोगात्मक)
└── packages/
    ├── xian-vl/          # ⚙️ कोर LLM/ASR ऑर्केस्ट्रेशन इंजन और कॉन्टेक्स्ट मैनेजर्स
    └── shared-types/     # 📦 कैनोनिकल मॉडल, कॉन्स्टेंट्स और शेयर्ड टाइप्स
```

> [!WARNING]
> **ASR / ऑडियो सीमा**: Lemonade पर लाइव ऑडियो स्ट्रीम अपलोड भेजना वर्तमान में सर्वर-साइड बैकएंड सीमाओं के कारण टूट गया है। परिणामस्वरूप, लाइव स्पीच ट्रांसलेशन सुविधाएं (जैसे **रेड मोड**) अक्षम हैं। सभी विज़ुअल OCR, टेक्स्ट ट्रांसलेशन, डिक्शनरी और चैट ग्राउंडिंग सुविधाएं पूरी तरह से चालू हैं।
